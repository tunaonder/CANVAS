CANVAS is developed as part of Sait Tuna Onder's M.S Thesis at Virginia Tech. The project is supervised by Dr. Osman Balci. The project title is "A Cloud-based Visual Simulation Environment for Traffic Networks". CANVAS offers a set of tools for the composition and visualization of simulation models for the traffic network problem domain. CANVAS uses an asynchronous visualization protocol with efficient resource utilization on the server, enabling concurrent usage of the IDE. CANVAS enables visualization of execution-heavy simulation within web browsers.
