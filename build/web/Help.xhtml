Requirements

CANVAS requires minimum screen resolution of 1024×768. CANVAS is not compatible with mobile devices.


Simulation Model Composition

Users can create multiple paths to define a traffic network model. Forks and Merges can be used to define connections among different paths. Currently, CANVAS provides following components: Enter Points, Exit Points, Move Spots, Traffic Lights, Forks and Merges. A Path must start with an enter point and end with an exit point. Move Spots and Traffic Lights can be added along the way.

If there is an alternative path on a specified point, select the "Convert To Fork" option and click on the move spot. The move spot will be converted into a Fork after the "New Path Probability" is defined.

Please use the "Convert To Merge" option when two different paths intersect into one path.

Example Model 1:

The provided model can be created with the following instructions:

1) Select the "Enter Point" to add the first component to the right edge of the map. Set Random Time Mimimum and Random Time Maximum to define vehicle generation timing.

2) Select the “Move Spot” component to add a move spot to the left-side of the enter point.

3) Select the “Traffic Light” component to add a traffic light. Enter the required parameters: Green Start Time, Green Duration, Red Duration.

4) Select the “Move Spot” component to add a move spot to the left-side of the traffic light.

5) Add an exit point to the left edge of the map.

Example Model 2:

This example shows how to convert move spots to the fork and merge objects.

The provided model can be created with the following actions:

1) Add an enter point by setting the Random Time Minimum and Random Time Maximum.

2) Add a move spot.

3) Add a move spot.

4) Add an exit point.

5) Select the “Convert to Fork” option in the model composer. Click on the move spot that is added with action 2. Enter the “New Path Probability”.

6) Add a move spot.

7) Add an exit point

8) Add an enter point by setting the Random Time Minimum and Random Time Maximum.

9) Add a move spot.

10) Select the “Convert to Merge” option in the model composer. Click on the move spot that is added with action 3.
